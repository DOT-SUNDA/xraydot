- via WGET
```
bash -c "$(wget -qO- https://raw.githubusercontent.com/givpn/aio/master/xray)"
```
- via CURL
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/givpn/aio/master/xray)"
```
